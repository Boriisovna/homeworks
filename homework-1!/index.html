GIT-шпаргалки

  • git config --global user.name blabla-встановлює імя користувача(глобально-для всіх проектів)
  • git config --global user.email blabal@bla-встановлює Email користувача (глобально)
  • git init - ініціалізує репозиторій в папці, де виконується команда
  • git status- показує стан файлів в робочому каталозі
  • git add- додає файли робочого каталогу у індекс;
  • git commit- зберігає змінені файли
  • git log-виводить історію комітів
  • git checkout-переключення на іншу гілку
  • git branch- команда для створення або керування гілками
  • git push- закидає файли в віддалений репозиторій
  • git pull-загружає зміни з віддаленого репозиторію в локальний
  • git merge- використовується зазвичай для обєднання гілок
  • git clone- скачує репозиторій разом зі всією історією змін

    • Що таке .gitignore файл?

    Використовується для того, щоб приховати файли та папки від git.

  • git revert- скасовує коміт, створюючи новий з протилежним змістом
  • git reset- відміна локальних приватних змін
  • git rebase- обєднує зміни, зроблені в одній гілці, з другою гілкою
  • git cherry-pick-переносить певні коміти з одного місця репозиторію в інший
  • git stash- тимчасово зберігає всі незафіксовані зміни в окреме сховище
